How to correct or update data

This has changed. Editing the database (PocketBase) directly is no longer the way.

Do not edit PocketBase directly. Changes made there are untracked, the system does not learn from them, and they can be overwritten on the next update.

Instead:

• To add data, upload the document through the uploader.

• To correct or ask about data, use Ask MIDD. Your correction is applied through the system, so it is tracked and the system improves.
