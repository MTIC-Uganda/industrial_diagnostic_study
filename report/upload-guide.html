How to add a document

This has changed, for the better. You no longer touch GitHub and you no longer name files a special way.

Use the Document Uploader: pick the value chain, describe in your own words what the document is and what should be done with it, attach the file, and submit. The system extracts the data, stores it, and rebuilds the dashboard within a few minutes. Duplicate documents are detected automatically.
