# Annex A: Value Chain Status Reports (Sector Snapshots)

These one-page **sector snapshots** distil the current status, market, constraints, priorities and green opportunity for each value chain in this report. Each is designed to be **extracted and shared independently** as a quick-reference brief. Figures and sources are detailed in the corresponding chapter (4–6) and the consolidated References.

## A.1 Iron & Steel — Status Snapshot

| Indicator | Value (2024/25) |
|---|---|
| Installed capacity / utilisation | **1,682,779 tpa / 30.5%** |
| Iron-ore reserves | ~350 million tonnes |
| Sponge-iron (Phase II) | ~400,000 tpa (Tembo Iganga 350k; Abyssinia Jinja 30k) |
| Industry investment | > USD 1 billion |
| Direct jobs | ~2,500 (+ large indirect) |
| Largest import | Flat steel (HS 7208) — **USD 219.5m** |
| Competitive export | Rebar (HS 7214) — **USD 55.8m** |
| Cost penalty | Imported-scrap billet ~USD 676/t vs gas-DRI ~USD 478/t |

- **Position:** strong downstream (rolling, ~70% of capacity); critical gap upstream (ironmaking/steelmaking); exports ore raw.
- **Top constraints (weighted):** upstream feedstock gap (5); electricity reliability & cost (5, ~8–10¢ vs peers); market/standards & import competition (4); capital for upstream (4).
- **Priorities:** ① Ironmaking (DRI on domestic ore) ② Flat-steel production ③ Long products (defend & expand) ④ Iron-ore mining & beneficiation.
- **2040 (tenfold):** demand ~1 Mtpa → **~5.5–8 Mtpa**.
- **Green edge:** gas/H₂-DRI + hydro-EAF (also cheapest); slag → cement.

## A.2 Copper & Allied Metals — Status Snapshot

| Indicator | Value |
|---|---|
| Kilembe reserves | >6.5 Mt ore @ 1.77% Cu; ~5.5 Mt @ 0.17% cobalt |
| Mine status | Dormant since 1980s; **production-sharing agreement Mar 2025** |
| Largest import | Insulated cable (HS 8544) — **USD 57.7m** (rising) |
| Feedstock imports | Cathode (7403) USD 6.9m + wire (7408) USD 3.1m |
| Lead fabricator | Cable Corporation (est. 1968) — **~65% market** |
| Demand driver | Grid access only **25.3%** (UBOS 2024) |

- **Position:** dormant midstream (mining/refining); strong, competitive downstream cable industry running on imported metal.
- **Top constraints (weighted):** dormant midstream/feedstock (5); capital & long lead time (5); energy reliability (4).
- **Priorities:** ① Insulated wire & cable (expand) ② Kilembe cathode & cobalt (revive) ③ Copper wire rod/semi-fabrication.
- **2040 (tenfold):** cable demand several-fold on electrification → order of **USD 300–500m+/yr**.
- **Green edge:** hydro electro-refining; cobalt for batteries; tailings reprocessing; scrap recycling.

## A.3 Automotive — Status Snapshot

| Indicator | Value |
|---|---|
| Flagship | Kiira Vehicle Plant, Jinja — **commissioned 27 Sep 2025** |
| Capacity | ~2,500 veh/yr → target 10,000 |
| Vehicle + parts imports | **> USD 650m/yr** |
| Used vehicles imported | ~48,700/yr |
| Motorcycle imports (HS 8711) | USD 157.1m |
| KMC financing gap | ~UGX 50bn (FY25/26) |

- **Position:** early, state-anchored assembly (KMC + motorcycle assembly); market dominated by used imports.
- **Top constraints (weighted):** finance/capital for scale-up (5); shallow local content (4); demand policy / used-import competition (4).
- **Priorities:** ① Buses / EV buses (KMC) ② Motorcycle assembly (CKD → e-boda) ③ Spare-parts & component localisation.
- **2040 (tenfold):** multi-fold mobility demand — capture via domestic assembly vs. the USD 650m+ import leak.
- **Green edge:** electric buses (Kayoola) and e-boda on the hydro grid; battery link to Kilembe cobalt.
