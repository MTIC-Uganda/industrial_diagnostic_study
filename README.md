# Industrial Diagnostic Study

**Client:** Ministry of Trade, Industry and Cooperatives (MTIC), Uganda
**Programme:** Uganda-UNIDO Programme for Country Partnership (PCP) for Inclusive and Sustainable Industrial Development (ISID)
**Deadline:** 8 June 2026

## Objective

Produce a high-level industrial diagnostic report covering all 9 NDP IV priority manufacturing value chains. For each chain: map the full value chain, assess the accessible market, and identify the products Uganda should prioritize.
